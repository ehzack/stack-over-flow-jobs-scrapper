## Features

- puppeteer
- Node Js

`NB` please make sure you're running node version v14.1.0 or higher.

## Installation

### Step 1 – Prerequisites

```bash
npm install
```

## Step 2 – Start

### options:

- -p : page count
- -l : country
- -f : fileName

```bash
npm run jobs  -p 15  -l united\state -f uk-jobs.csv
```
